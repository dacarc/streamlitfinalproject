# 📊 Income & Work Analysis Dashboard

## 🌐 Live App
👉 https://income-work-analysis.streamlit.app

---

## 📌 Project Overview

This project is an interactive data dashboard built with Streamlit that analyzes income and work patterns using IPUMS survey data.

The goal of this project is to transform raw labor data into clear, interactive insights that are easy to explore and understand.

Through this dashboard, users can:
- Explore how income varies by age  
- Compare work efficiency across telework and non-telework employees  
- Interact with filters to dynamically update visualizations  

This project demonstrates my ability to clean data, engineer features, and build interactive data applications using Python.

---

## 📊 Key Features

- Interactive Streamlit dashboard  
- Telework vs non-telework filtering  
- Income trends by age  
- Hourly efficiency distribution analysis  
- Real-time updates based on user input  

---

## 🛠️ Tools Used

- Python

## 📁 Dataset
